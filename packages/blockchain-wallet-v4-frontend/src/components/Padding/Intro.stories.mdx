# Padding

---

This component is build to provide a simple component to add padding to an section

## Example

This example is adding a padding of 10px on all directions

```jsx
<Padding.Symetric top={10} left={10} bottom={10} right={10}>
  <span>Padding content</span>
</Padding.Symetric>
```

## Symetric

You can use the helper component to simplify how to create symetric padding

```jsx
<Padding.Symetric horizontal={16} vertical={10}>
  <span>Padding content</span>
</Padding.Symetric>
```
