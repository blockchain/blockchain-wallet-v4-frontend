# useFetcher

The fetcher hook should be used when you need to fetch JSON data from an endpoint.

To use it, you need to provide the URL endpoint and the result data, from there the useFetcher will handle the loading, fetching, error and success states

## Example

```ts
type DATA = {
  // ...api return type
}

const { data, error, isLoading, isFetching } = useFetcher<DATA>('http://example.com')
```
